# myGolangCodes

存放本人阅读golang项目遇到好的一些处理方法及自己实现的一些小功能,为后面方便使用

## 目录
1. Concurrent 并发
2. http-request 一个解析数据包,可以用来做poc捕获的工具
3. some-function 一些阅读golang项目 拿下来的较常用的方法
4. xray-subdomain-extraction xray 实现的子域名解析
5. json-decode json文件解析的例子
